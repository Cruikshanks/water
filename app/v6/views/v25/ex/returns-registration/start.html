Create an account to manage your water abstraction licence online

You need to create an online account to access your licence information.

Impoundment licences are not yet part of this service.

We will send you a security code by post to complete this process. This should take around 5 working days.

If you already have a security code it should take you about 5 minutes to create an account.

Before you start

You will need:

  • an email address
  • your licence numbers
  • your security code (if you have one)
