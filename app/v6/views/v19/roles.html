Water resource licensing service roles

Super user

{% include "v19/roles-nav/nav-admin-view.html" %}

The super user can do everything.

Data & Returns

{% include "v19/roles-nav/nav-officer-head.html" %}

Data & Returns users can view licences, see versions of licences and see reports about notifications that have been sent.

Returns entry

{% include "v19/roles-nav/nav-returns-head.html" %}

Returns entry users can view licences and enter returns data.

Abstraction reform operator

{% include "v19/roles-nav/nav-reform-head.html" %}

Abstraction reform operator can view licences and they can see a list of licence records so that they may compare these with NALD data and make edits.

Abstraction reform approver

{% include "v19/roles-nav/nav-reform-head.html" %}

Abstraction reform approver can view licences and they can see a list of edited licence records so that they may approve them or send them back to editors for more edits.

Environment Officer

{% include "v19/roles-nav/nav-officer-head.html" %}

The Environment Officer can view licences, send notifications to licence holders and they can also see reports.
