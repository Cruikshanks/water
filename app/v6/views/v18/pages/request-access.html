Request to view other people’s licences

If you manage licences for other people (as an agent or consultant, for example), you can use this service to view information about those licences online.

How does the request process work?
  1. We'll ask you for licence holder's email address and licence number(s).
  2. We'll email the licence holder to ask permission to share their licence information with you online. The licence holder can accept or reject your request.
  3. If they accept, we'll send them a security code to the postal address on their licence to make sure they are the registered licence holder. They should receive the code within 5 working days.
  4. Once the licence holder receives the code, they can either enter it into our system or ask you to do it for them. Please contact them to agree the best way for this to happen.
  5. As soon as the security code has been entered, you'll be able to view the licence information online.
