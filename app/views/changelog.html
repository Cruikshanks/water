Prototype change log

This summarises the main changes we’ve made in response to feedback.

Version 1a to 1b

Version 1 was our first version prototype.

1a was user tested. 1b was formed as a second iteration of version 1, in responce to user testing.

The table below lists the changes between version 1a and 1b following user feedback.

Summary of change in v1

Reason/source

Changed header to sentence case on online licence page

Grammar correction

Alignment change from two column table list, to left aligned single column list on online licence page

Users had trouble understanding the information on the page

Name and site address added on online licence page

Internal user feedback states this information is useful for site visits

Other contact details link moved up the page, next to associated content on online licence page

Feedback suggests links were not easy to spot in their previous positions

Abstraction point grid ref - title change, and data format change on online licence page

Internal users expected to see SJ or SK grid references

Abstraction amount label change on online licence page

Mirroring the format on the paper licence to make amounts clearer

Hands off flow restriction added to online licence page

Internal users noted this is important information

Changed header to sentence case on terms of use page

Grammar correction

Changed header to sentence case to contact details page

Grammar correction

Changed ‘Business details’ to ‘Abstraction return contact details’ on contact details page

Internal user feedback states that returns have a separate address, no user research has suggested that business address is required.
